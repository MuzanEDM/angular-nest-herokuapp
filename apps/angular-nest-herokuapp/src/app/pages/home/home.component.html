Fullstack Development

All included end-to-end application

From code to the deployment, this project aims to be a point of reference for many ideas and questions - make a suggestion!

  • Open-source

    All of the code and configuration of this website can be found on GitHub here and can be forked and extended by anyone - everything is open.

  • Nx Monorepo tooling

    An Nx Workspace is how all of the code of this website is managed and organized. Any change that affects either the client or server is identified.

  • Angular

    The face of this website is powered by Angular 9 and has modules and components generated via the Angular CLI - more examples coming soon!

  • NestJS

    NestJS has any Angular developer feel at-home on the backend. NestJS is a Node.js framework that makes fullstack work enjoyable (especially with Angular!)

  • Heroku

    Connect your forked repo to a new Heroku app and begin to deploy easily. Heroku allows any developer to quickly get their websites published.

  • Tailwind UI

    Tailwind CSS provides the classes developers can use to achieve their designs. Tailwind UI provides developers designs and initial UI ideas.

  • GitHub Actions

    Pull requests towards this project are automatically tested for errors via GitHub Actions - issues are caught earlier rather than later.
